## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-is-available`
- `allow-search`
- `allow-install`
- `allow-uninstall`
- `allow-list-installed`
- `allow-list-outdated`
- `allow-upgrade`
- `allow-cancel-operation`
